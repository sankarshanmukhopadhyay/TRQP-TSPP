# TRQP-TSPP v0.6.0

This minor release fixes cross-repo version drift, documents the schema layout to prevent
future duplication, and adds Ayra Trust Network guidance.

## Highlights

- **Version pin fix**: `docs/hub-crosswalk.md` now agrees with `al-contract.json` on the
  Assurance Hub version (v0.9.0). Previously the crosswalk document lagged one release
  behind the machine-readable contract.
- **Schema layout documented**: new `schemas/README.md` explains the canonical
  (`schemas/core/`) vs alias (`schemas/`, `harness/schemas/`) layout, making it clear
  where to make normative changes and preventing future copy/paste duplication.
- **Ayra profile guidance**: `docs/profiles/ayra-baseline.md` maps TSPP controls to the
  Ayra Trust Network conformance tiers (Basic / Cross-Ecosystem / Sovereign) and lists
  the evidence artifacts to submit to the Ayra conformance process.

## No breaking changes

This release contains no changes to control IDs, assurance level semantics, test logic,
OpenAPI contract, or JSON Schema content. It is safe to adopt without updating dependent tooling.
