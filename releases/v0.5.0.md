# TRQP-TSPP v0.5.0

This release hardens the repository for higher-assurance deployments.

## Highlights

- Add executable AL3 and AL4 harness coverage for independent assessment, change control, rollback, policy, monitoring, key protection, and audit-log evidence.
- Expand CI to run AL1 through AL4 against the reference SUT and add unit tests for core validation helpers.
- Add AL contract pin verification using a canonical assurance-level snapshot and synchronize public-facing documentation with current cross-repo versions.
