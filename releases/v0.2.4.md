# TRQP TSPP v0.2.4

Release date: 2026-03-02

## What changed

This release improves interop guidance for GRID-style registrar directories by:
- Updating crosswalk documentation to the Assurance Hub GRID profile and verifier workflow
- Adding concise operator notes on treating directory data as claims, requiring signed feeds, and applying AL-based eligibility

## Compatibility

- No breaking changes.
- No protocol behavior changes; documentation and interop guidance only.
